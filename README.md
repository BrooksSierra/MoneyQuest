# MoneyQuest

MoneyQuest is a gamified personal finance tracker. It helps users manage income, expenses, and savings while earning XP and leveling up as they achieve financial goals.

---

Features

- Leveling System: Gain XP for logging expenses, saving money, and completing financial â€œbattles.â€

- Spend Tracker: Log expenses and view a categorized summary.

- Budget Setup: Set needs/wants/savings percentages.

- Battles: Create savings goals and earn XP for contributions.

- Spending Summary: View current and archived expenses with category percentages.

- Data Persistence: User progress is saved in a JSON file.
